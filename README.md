## 入门指南
首先，运行开发服务器：
```bash
Copy code
npm run dev
# 或者
yarn dev
# 或者
pnpm dev
# 或者
bun dev
```
 使用浏览器打开 http://localhost:3000 来查看结果。


您可以通过修改 app/page.tsx 来开始编辑页面。页面会在您编辑文件时自动更新。

此项目使用 next/font 来自动优化和加载 Inter，一个自定义的 Google 字体。
